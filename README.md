## Running locally

Install [LÃ–VE](https://love2d.org/) (also see [Getting Started](https://love2d.org/wiki/Getting_Started)).

Clone the project.

```bash
git clone git@github.com:IndaPlus22/vprytz-sorting.git
```

Assuming `love` is in your path, run:

```bash

```
